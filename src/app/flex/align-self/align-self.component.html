align-self: flex-start | flex-end | center | baseline | stretch

Allows a single item to have an alignment that is not the same as other items and can override the align-items attribute.
The default value is auto, which means it is equal to:
-the parent-element's align-items attribute,
-stretch if there is no parent element.
