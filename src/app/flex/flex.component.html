container attributes:

flex-direction flex-wrap flex-flow justify-content align-items align-content

item attributes:

order align-self flex-grow flex-shrink flex-basis flex

Notice:
  • when the flex-direction is a column, justify-content changes to the vertical and align-items to the horizontal.
  • Any container can be specified as a flex layout.
  • The float, clear, vertical-align attributes in child element would not work after the layout is set to flex.
